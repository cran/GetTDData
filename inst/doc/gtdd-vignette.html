Downloading Data from Tesouro Direto

Marcelo Perlin

2016-02-10

Introduction

Information regarding prices and yields of bonds issued by the Brazilian government can be download manually as excel files from the Tesouro Direto website. However, it can be painful to aggregate all of this data into something useful for analysis as the files don’t have an uniform format and are all divided by year and asset code.

The package GetTDData makes the process of importing data from Tesouro direto much easier. All you need is the name of the asset (e.g. LTN, LFT, NTN) and the desired date of maturity (e.g. 010116).

How to use GetTDData

Using GetTDData is pretty simple. Suppose you wanted data (prices and yields) for a LTN1 with a maturity date equal to 2016-01-01. For that, all you need to do is run the following code:

library(GetTDData)

asset.code <- 'LTN'   # File Name of asset 
maturity <- '010116'  # Maturity date as string (ddmmyy)

my.flag <- download.TD.data(asset.code = asset.code)

Now, let’s plot the prices to check if the code worked well.

library(ggplot2)

p <- ggplot(data = my.df, aes(x = as.Date(ref.date), y = price.bid, color = asset.code))
p <- p + geom_point() + scale_x_date() + labs(title = '', x = 'Dates')
print(p)

As expected, the prices from this bond will converge to its face value, 1000 R$, at the maturity date (2016-01-01). We can also look at its time-varying yield:

p <- ggplot(data = my.df, aes(x = as.Date(ref.date), y = yield.bid, color = asset.code))
p <- p + geom_point() + scale_x_date()+ labs(title = '', x = 'Dates' )
print(p)

The functions in GetTDData also works with multiple arguments as asset.code and maturity. Suppose we wanted to look at the prices of all available maturities for bonds of type LTN. All we need to do is to add the value NULL to the argument maturity:
